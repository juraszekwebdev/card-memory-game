# Card Memory Game
Simple Card Memory Game written in ReactJS

Demo version: [LINK](https://pj-cardmemorygame.netlify.app)

## Running project

### Install dependencies
### `yarn install`

### Run locally
### `yarn start`

### Build app
### `yarn build`
Builds the app for production to the `build` folder.

## Used technologies
- ReactJS
- Lodash
- SCSS
